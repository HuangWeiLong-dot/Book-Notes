# as if
##### 词义
- 美：/ æz ɪf /；
- #词性/conj  好像，仿佛
##### 例句
- I don't feel as if I belong here.
	- 我在这里感觉格格不入。
- I felt as if my heart had stopped.
	- 我觉得好像我的心都不跳了。
- I ate as if there was no tomorrow.
	- 我狼吞虎咽地吃着。
