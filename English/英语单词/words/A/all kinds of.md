# all kinds of
##### 词义
- 美：/ ɔl kaɪndz əv /；
- #词性/adj  各种各样的：许多不同种类的事物。
##### 例句
- They sell all kinds of things.
	- 他们出售各种各样的东西。
- Adoption can fail for all kinds of reasons.
	- 收养可能因为各种各样的原因失败。
- She enjoys all kinds of music, as I do.
	- 她和我一样什么音乐都喜欢。
