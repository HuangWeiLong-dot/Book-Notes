# run away
##### 词义
- 美：/ rʌn əˈweɪ /；
- #词性/v  逃跑，离家出走，或者是逃脱
##### 例句
- His first instinct was to run away .
	- 他的本能反应就是逃跑。
- I can't stand any more. I'm going to run away .
	- 我再也忍受不了了，我打算逃跑。
- They never became tame; they would run away if you approached them.
	- 它们从没有被驯服，如果你靠近，它们就跑开了。
