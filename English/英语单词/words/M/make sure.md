# make sure
##### 词义
- 英：/ meɪk ʃʊə(r) /；美：/ meɪk ʃʊr /；
- #词性/v  确保；让每个人都知道
##### 例句
- Make sure the wires don't touch.
	- 一定不要让金属线搭在一起。
- Make sure that you are not underinsured.
	- 确保你的保险是足额的。
- Make sure that you defrost it completely.
	- 确保将其彻底除霜。
