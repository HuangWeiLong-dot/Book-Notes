# so that
##### 词义
- #词性/conj  以便，为了；如此……以致于……
##### 例句
- Bend the wire so that it forms a 'V'.
	- 把铁丝弯成V形。
- Can you work it so that we get free tickets?
	- 你能不能为我们搞到些免费票？
- Bess spoke up, smiling so that her dimples showed.
	- 贝丝大声发言，微笑着，使脸上露出了两个酒窝。
